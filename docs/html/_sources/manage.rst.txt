manage Module
=============

The **`manage`** module provides the following commands from Django's `manage.py` file:

- `runserver`: Starts the development server.
- `migrate`: Applies database migrations.
- `makemigrations`: Generates new migration files.
- `createsuperuser`: Creates a new superuser account.
- `collectstatic`: Gathers static files for deployment.
- `test`: Runs the test suite.

For more information about each command, refer to the  
`Django official documentation <https://docs.djangoproject.com/en/stable/ref/django-admin/>`_.
