manage Module

This module accepts arguments from the command line and provides the following commands from Django’s manage.py file:

  • runserver: Starts the development server.

  • migrate: Applies database migrations.

  • makemigrations: Generates new migration files.

  • createsuperuser: Creates a new superuser account.

  • collectstatic: Gathers static files for deployment.

  • test: Runs the test suite.

For more information about each command, refer to the Django official documentation.

Django’s command-line utility for administrative tasks.

manage.main()[source]

Loads environment variables from a .env file in this project’s route directory.

Environment Variables:

SECRET_KEY: The value to use for this Django project’s SECRET_KEY. DJANGO_SETTINGS_MODULE: The module path to this project’s settings module.
